📊 JobPulse – Data Transformation & Dashboard Project

📌 Overview

This project focuses on cleaning and transforming the Wuzzuf job postings dataset using Power BI Power Query.
The goal is to prepare the data for analysis and build a dashboard to explore job trends and skills.

📂 Dataset
Source: Wuzzuf Jobs Posting dataset
File: Wuzzuf-Jobs-Posting.csv
Rows: 25,000+
Includes data like job title, skills, experience, and number of applicants

⚙️ Data Processing (Power Query)

The dataset was cleaned and transformed using Power BI:

Removed blank rows
Handled missing values in numeric columns
Filtered out null values in important columns
Cleaned "Job Skills" column (removed brackets and symbols)
Split skills into separate rows
Trimmed and cleaned text data
Created a separate Skills table
Extracted Year, Month, and Year-Month from dates

🧱 Data Model

Two main tables were created:

Wuzzuf-Jobs-Posting (main dataset)
Skills (for skill-level analysis)

📊 Dashboard Features
Job trends over time
Most required skills
Number of applicants analysis
Experience level insights

🛠️ Tools Used
Power BI
Power Query
Excel / CSV

📈 Project Output
Cleaned dataset ready for analysis
Interactive Power BI dashboard
